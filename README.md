# Semáforo con Py

## Posibles Funciones que podemos hacer que haga el semáforo:
* Cambiar Luz
``` 
ejemplo:
    Rojo a Verde
    Verde a Amarillo
    Amarillo a Rojo
```
* Set intervalos de tiempo para una luz
``` 
ejemplo:
    10 SEGUNDOS VERDE
```
